# Topology — Master System Design Visually

An interactive platform for learning and practicing system design concepts through visual diagrams, quizzes, and real-world case studies.

## Features

- **Fundamentals** — Core system design concepts with interactive demos
- **Building Blocks** — Reusable components like caches, queues, and databases
- **Patterns** — Common architectural patterns with visual explanations
- **Case Studies** — Real-world system breakdowns
- **Canvas** — Drag-and-drop system design whiteboard powered by ReactFlow
- **Estimate** — Back-of-the-envelope capacity estimation calculator
- **Interview** — Practice mode for mock system design interviews

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** — build tool and dev server
- **Tailwind CSS** — utility-first styling
- **ReactFlow** — interactive node-based diagrams
- **D3.js** — data visualizations
- **Framer Motion** — animations
- **Zustand** — state management
- **React Router v6** — client-side routing

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start dev server |
| `npm run build` | Type-check and build for production |
| `npm run preview` | Preview production build locally |
| `npm test` | Run unit tests with Vitest |
